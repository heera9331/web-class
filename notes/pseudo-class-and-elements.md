# Pseudo class and elements

A pseudo-class is used to define the special state of an element.

```css
selector:pseudo-class {
  property: value;
}
```

1. `hover`

```css
a:hover {
  color: blue;
}
```

2. `:focus`

```css
input:focus {
  border-color: green;
}
```

3. `:nth-child(n)`

```css
li:nth-child(2) {
  font-weight: bold;
}
```
